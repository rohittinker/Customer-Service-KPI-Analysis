## 📞 Call Centre Performance Analytics Dashboard
### 📝 Project Overview
    This project involves the analysis of a dataset containing 1,000+ customer care records. The goal was to transform raw operational data into actionable insights  regarding agent performance, customer satisfaction, and revenue trends using Excel 2019.

### 📊 Dashboard Preview
### 🚀 Key Insights & KPIs
    Using advanced Excel functions, I tracked and visualized the following critical metrics:
    
    Customer Satisfaction (CSAT): Achieved an average rating of 3.9/5.0.
    
    Operational Volume: Successfully processed and analyzed 1,000 calls.
    
    Financial Impact: Total amount generated was $96.62K against a target of $20.87K.
    
    Trend Analysis: Identified significant call volume peaks in March and October.
    
    Performance Benchmarking: Compared 5 representatives (R01-R05) based on call count and revenue generation.

### 🛠️ Technical Skills Demonstrated
    Data Cleaning: Standardized currency formats, handled durations, and categorized caller demographics by City and Gender.
    
    Pivot Tables & Slicers: Utilized Pivot Tables for data aggregation and Slicers for interactive filtering by Representative and City.
    
    Data Visualization: Created dynamic charts (Line, Bar, and Doughnut) to represent complex trends clearly.

### 📂 How to Use
1. Download the .xlsx file from this repository.
    
2. Open in Microsoft Excel (Optimized for Excel 2019).

3. Use the Slicers on the dashboard to interactively filter data and view specific representative performance.
